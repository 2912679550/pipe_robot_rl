## Gazebo仿真初探——四连杆并联模型仿真实践

由于SolidWorks导出的urdf模型不支持并联机构，而105管道机器人主要的夹紧、变形机构都是并联方案，为此需要探明如何合适地处理这个问题。查阅资料了解到，由于ROS与Gazebo是分别相对独立地发展的，因此在Gazebo原生支持中，SDF是较为高级的一种模型。它好像是支持闭链结构的。对此重点研究。
